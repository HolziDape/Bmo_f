# PWA (Progressive Web App) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make both BMO web apps installable as PWAs by adding a minimal Service Worker and completing missing PWA assets in the Freund app.

**Architecture:** Add a `/sw.js` Flask route (pass-through fetch SW) to both apps and wire up SW registration in HTML. The Freund app additionally needs the `BMO_SVG` icon constant, `/icon.svg`, and `/manifest.json` routes plus the HTML meta tags already present in the Admin app.

**Tech Stack:** Python/Flask, JavaScript Service Worker API, Web App Manifest (JSON)

- [ ] **Step 1: Add `/sw.js` route**

Open `D:\python\scripts\Bmo_main\src\bmo_web.py`. After the `manifest()` function (which ends at line 3179 with `)`), insert:

```python
@app.route('/sw.js')
def sw_js():
    js = (
        "self.addEventListener('install', () => self.skipWaiting());\n"
        "self.addEventListener('activate', e => e.waitUntil(clients.claim()));\n"
        "self.addEventListener('fetch', e => e.respondWith(fetch(e.request)));\n"
    )
    return Response(js, mimetype='application/javascript')
```


- [ ] **Step 2: Add `/icon.svg`, `/manifest.json`, `/sw.js` routes**

After the `api_lite_mode()` function (which ends at line 2154 with `return jsonify(lite_mode=None)`), insert these three routes:

```python
@app.route('/icon.svg')
def icon_svg():
    return Response(BMO_SVG, mimetype='image/svg+xml')

@app.route('/manifest.json')
def manifest():
    return jsonify(
        name="BMO",
        short_name="BMO",
        start_url="/",
        display="standalone",
        background_color="#0f172a",
        theme_color="#5eead4",
        icons=[{"src": "/icon.svg", "sizes": "any", "type": "image/svg+xml"}]
    )

@app.route('/sw.js')
def sw_js():
    js = (
        "self.addEventListener('install', () => self.skipWaiting());\n"
        "self.addEventListener('activate', e => e.waitUntil(clients.claim()));\n"
        "self.addEventListener('fetch', e => e.respondWith(fetch(e.request)));\n"
    )
    return Response(js, mimetype='application/javascript')
```

The block after the insertion should be `@app.route('/api/settings', methods=['GET'])`.
